# CodeReader 插件

<!-- Plugin description -->
CodeReader 是一款纯粹的、用于程序员上班摸鱼的、支持本地文件上传的、JetBrains IDE 小说阅读插件。

欢迎使用 CodeReader 插件！本插件可让您直接在 IDE 的状态栏中阅读 `.txt` 和 `.epub` 格式的书籍，提供一种在编码和阅读之间无缝切换的方式，而无需离开您的开发环境。

## 功能

- **在状态栏中阅读**: 在状态栏中逐页显示您的书籍内容。
- **支持 `.txt` 和 `.epub`**: 从纯文本和 EPUB 文件加载书籍。
- **高效的 `.txt` 和 `.epub` 文件处理**: 采用优化的文件指针索引与懒加载策略，即使是 GB 级大型文件也能实现瞬时加载和极低内存占用的流畅阅读。
- **智能分段显示**: EPUB 文件支持段落感应，通过长空格自动标识段落切换，并优化分页边缘留白，阅读体验更连贯。
- **极致性能**: 解耦内容更新与 UI 渲染逻辑，即使在高频自动翻页下也依然丝滑，CPU 占用极低。
- **阅读历史**: 自动记录您的阅读历史，包括小说名称、章节、进度和当前页面内容。历史记录在 IDE 关闭时清除。
- **高度隐蔽与极致稳定的显示**: 
  - **无抖动显示**: 引入固定宽度显示区域，彻底解决翻页和章节数字变化时状态栏频繁伸缩、抖动的问题。
  - **100% 原生融合**: 针对 100% 可见度使用系统原生颜色对象，确保与 IDE 状态栏文字完全一致，支持主题即时切换。
  - **完整内容呈现**: 优化了长文本截断逻辑，确保每一页内容完整无遗漏，拒绝 `...` 截断带来的阅读中断。
  - **布局解耦**: 章节信息与阅读内容独立显示，开启/关闭章节信息不再推挤正文位置，提供极致稳定的摸鱼体验。
  - **可见度调节**：无缝调节文字透明度，实现极致隐蔽。
- **文件导入兼容性**: 优化了文件选择器逻辑，支持在所有 JetBrains 平台（包括最新版 GoLand）中正确置灰不支持的文件。
- **全中文界面**: 菜单和提示信息全面中文化，操作更直观。

## 如何使用

### 加载书籍

- **从菜单**: 前往 `工具 -> CodeReader` 并选择 `导入本地文件 (txt/epub)`。

### 阅读

- **下一页**: `alt + meta + RIGHT`
- **上一页**: `alt + meta + LEFT`
- **开启/暂停自动翻页**: `alt + meta + K`

### 阅读历史

- **查看历史**: 前往 `工具 -> CodeReader -> 查看阅读历史` 以查看您最近阅读的书籍列表。如果历史记录为空，将显示提示消息。

### 显示选项

- **切换阅读器可见性**: `alt + meta + H` 或 直接点击状态栏中的插件主体 以显示或隐藏状态栏中的 CodeReader。
- **切换章节信息**: `alt + meta + J` 以显示或隐藏当前章节标题、章节进度和书籍进度（仅适用于 `.epub` 文件）。
- **修改字数**: 前往 `工具 -> CodeReader -> 修改单次显示字数` 以滑块形式调节显示字数，并支持实时预览效果。
- **调节可见度 (隐蔽度)**: 前往 `工具 -> CodeReader -> 调节显示效果 (隐蔽度)` 使用滑块微调文字的透明度。
- **自动翻页设置**: 前往 `工具 -> CodeReader -> 自动翻页设置` 调整翻页时间间隔 (0.1秒 - 5.0秒)。

### 章节选择

- **选择章节**: 对于 `.epub` 文件，您可以通过前往 `工具 -> CodeReader -> 选择章节` 来跳转到特定章节。

### 清理缓存

- **清理缓存**: 前往 `工具 -> CodeReader -> 清理缓存与历史` 以清除当前书籍的内容和您的阅读历史。

### 调整快捷键

- **调整快捷键**: 前往 `设置 -> Keymap -> 搜索 "CodeReader"` 以调整插件的快捷键。

## Q&A

- **Q: 插件支持哪些文件格式？**
  - **A**: 插件支持 `.txt` 和 `.epub` 文件格式。
- **Q：还会支持其他文件格式吗？**
  - **A**: 常用的小说文件格式还有 `AZW3`、`MOBI` 等。前者目前只有一个商用的库可以解析，怕有一些问题；后者完全是下位替代，不太好用不想做。
- **Q: 插件是否支持从网络上下载文件？**
  - **A**: 不支持。作者从来都是本地下载小说再阅读的（因为能找到高质量的书），网站上太过良莠不齐不喜欢用。

感谢您使用 CodeReader！我们希望它能帮助您保持高效并享受阅读。
